WEBSITE ACCESSIBILITY

Website accessibility is the practice of creating websites that are usable for all visitors regardless of disability or impairment. This means ensuring that websites are designed and coded in a way that people with disabilities can use them by adjusting format, structure, navigation, visuals, and written content. The World Health Organization estimates that 16% of the global population has a disability. By following the design and development guidelines provided on this website, you can ensure that your website is built to accomodate anyone.

Who does web accessibility affect?

Web accessibility supports all disabilities that affect access to the web including:

  • Auditory
  • Congnitive
  • Neurological
  • Physical
  • Speech
  • Visual

Why should I care about website accessibility?

Creating a website that is accessible not only supports those with disabilities, but also allows more users overall to access your site. Adhering to accessibility guidlines demonstrates a company's commitment to inclusivity and shows that you care about the user experiences.
