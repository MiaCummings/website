WEBSITE ACCESSIBILITY

Website accessibility is the practice of creating websites that are usable for all visitors regardless of disability or impairment. This means ensuring that websites are designed and coded in a way that people with disabilities can use them, allowing all visitors of the site to have a "seamless experience." Web accessibility supports the following disabilities that affect access to the Web:

  • Auditory
  • Congnitive
  • Neurological
  • Physical
  • Speech
  • Visual

Creating a website that is accessible not only supports those with disabiliites, but also allows more users overall to access your site.
